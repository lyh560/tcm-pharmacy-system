| 文件名 | 说明 |
|--------|------|
| login.png | 登录页面截图 |
| dashboard.png | 首页看板截图 |
| medicine.png | 药品管理截图 |
| sales.png | 销售管理截图 |
| purchase.png | 采购管理截图 |
| report.png | 报表统计截图 |
| user.png | 用户管理截图 |
| inventory.png | 库存管理截图 |

## 截图建议

1. 使用 1920x1080 或 1440x900 分辨率
2. 确保浏览器窗口最大化
3. 展示有数据的状态
4. 可以适当模糊敏感信息

## 如何添加截图

1. 运行项目并访问各页面
2. 使用截图工具（如 Snipaste、微信截图等）截图
3. 将截图保存到此目录
4. 提交到 GitHub
